## Technology Stack
- *Backend:* PHP 8.2, Laravel 11
- *Frontend:* NPM

## Installation
-composer install
-npm install
-npm run build
-npm run dev
-php artisan migrate
-php artisan serve

### Prerequisites
Make sure you have the following installed:
- PHP 8.2
- Composer
- Laravel 11
- MySQL 
- NPM
